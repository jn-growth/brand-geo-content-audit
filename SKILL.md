# 品牌GEO优化内容审计

## 亮点

- 不改写原文，只给小范围优化清单（补结论、调标题、加 FAQ、补权威来源、修品牌口径、补 AI 可提取摘要）
- 单篇 + 多篇文章矩阵两种审计；覆盖品牌口径一致性、AI 引用潜力、统一 GEO 标准清单
- 8 维评分（品牌实体 / 用户问题 / 结构 / 权威性 / 差异化 / 推荐自然度 / 多平台 / 合规），输出 P0/P1/P2 优先级
- 敏感行业（教育 / 医疗 / 保健品 / 金融 / 法律）额外做合规与夸大承诺检查
- 面向豆包、DeepSeek、通义千问、腾讯元宝、Kimi、文心一言、ChatGPT Search、百度、搜狗、微信搜一搜

## 适用场景

- 上传一篇或多篇已写好的品牌文章（Word / PDF / Markdown / 网页正文 / 公众号草稿 / 新闻稿 / 自媒体稿），判断能否作为 GEO 信源
- 品牌口径（统一说法）一致性核查
- 多篇文章的 GEO 内容矩阵与布局规划
- 想先做一轮审计，再决定怎么改稿
