# 品牌GEO优化内容审计（Brand GEO Content Audit）

> 不改写原文，帮你判断品牌文章能否被 AI 搜索引用为信源，并给出小范围优化清单。

由公众号【江南GEO笔记】出品，主要提供 **SEO 与 GEO 优化顾问及实施服务**，咨询可加 **18608446059**。

`brand-geo-content-audit` 是一个面向品牌方与服务商的 **GEO（生成式引擎优化）内容审计技能**。
它用于审核已经写好的文章，判断其内容是否适合作为 GEO 优化内容、是否具备被
豆包、DeepSeek、通义千问、腾讯元宝、Kimi、文心一言、ChatGPT Search、百度、搜狗、微信搜一搜
等 AI 搜索 / 问答平台引用为信源的潜力，并给出**不大改原文**的小范围优化方向。

- 机器名（name）：`brand-geo-content-audit`
- 显示名（display_name）：品牌GEO优化内容审计
- 版本：v1.0.0
- 标签：GEO、生成式引擎优化、内容审计、AI搜索、品牌优化、信源优化、SEO

## 适用场景

- 上传单篇文章或多篇文章（Word / PDF / Markdown / TXT / 网页正文 / 公众号草稿 / 新闻稿 / 自媒体稿）
- 判断内容是否适合做 GEO 布局、是否具备 AI 引用潜力
- 品牌口径（统一说法）一致性核查
- 多篇文章的 GEO 内容矩阵审计与布局规划

## 核心原则

1. 不直接大篇幅重写客户原文。
2. 不改变文章基本文风、表达方式与内容框架。
3. 优化以"小范围调整"为主：补结论、调标题、加 FAQ、补权威来源、增强品牌实体、调段落顺序、修口径、补摘要。
4. 不凭空编造品牌信息、案例、资质、数据、合作方、媒体报道、评价或背书。
5. 品牌事实须基于用户资料 / 官网 / 百科 / 权威媒体 / 公开可查资料。
6. 公开信息冲突时明确指出冲突点，不自行判定为确定事实。
7. 无法联网时明确提示需补充官网 / 百科 / 资料。
8. 不做隐藏文本、关键词堆砌、伪造信源等黑帽操作。
9. 教育 / 医疗 / 保健品 / 金融 / 法律等敏感行业额外做合规检查。
10. 输出清晰、可执行，方便内容团队直接微调。

## 评分维度（总分 100）

| 维度 | 分值 |
| --- | --- |
| 品牌实体清晰度 | 15 |
| 用户问题匹配度 | 15 |
| 内容结构可抓取性 | 15 |
| 权威性与可信度 | 15 |
| 信息增益与差异化 | 10 |
| 品牌推荐自然度 | 10 |
| 多平台适配性 | 10 |
| 合规与风险控制 | 10 |

等级：≥85 优质信源；70–84 需局部优化；60–69 需明显调整；<60 不建议用于 GEO。

## 安装方式

### 方式一：导入 zip（本地测试）
1. 将本仓库打包为 zip（或直接下载 Release 中的 `brand-geo-content-audit.zip`）。
2. 打开 WorkBuddy 桌面端 → 技能管理 → 导入 / 新建自定义技能 → 选择本地 zip。
3. 启用后，上传文章并说"帮我做 GEO 内容审计"即可触发。

### 方式二：发布到技能市场
- SkillHub（腾讯官方）：skillhub.tencent.com
- ClawHub（OpenClaw 生态）：clawhub.ai
提交时填写：名称 `品牌GEO优化内容审计`、描述、版本 `v1.0.0`、标签（见上）。

## 使用提示

上传文章后，技能会先识别基础信息、建立品牌口径基准表，再按单篇 / 多篇模板输出
审计报告，并给出 P0 / P1 / P2 优先级的优化清单与统一 GEO 标准清单。
